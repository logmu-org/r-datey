# The `datey` version of NA

The `datey` version of NA

## Usage

``` r
NA_datey_
```

## Format

An object of class `datey_type` (inherits from `datey`) of length 1.

## See also

[is_NA](https://logmu-org.github.io/r-datey/reference/is_NA.md),
[NA_durationy\_](https://logmu-org.github.io/r-datey/reference/NA_durationy_.md),
[integer_constants](https://logmu-org.github.io/r-datey/reference/integer_constants.md)

## Examples

``` r
  is_datey(NA_datey_) # TRUE
#> [1] TRUE
```
